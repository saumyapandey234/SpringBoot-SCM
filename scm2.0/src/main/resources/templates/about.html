The Story

Why was this
built?

It started as a simple question what if saving a contact was actually complete? Not just a name and number but email, social links, a photo everything in one place.

This project was never meant to be a product or a startup. It was built purely to learn how real applications work — Spring Security, OAuth logins, cloud storage, payment gateways — one feature at a time.

And along the way, it became something actually useful for friends. That's the whole point.

"The best way to learn is to build something real — even if it's just for your friends."
